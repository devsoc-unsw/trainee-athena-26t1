# Frontend
```bash
cd client
npm i
npm run dev
```

# Backend
Before you run the following commands, make sure you have the env file in the server folder:
```
cd server
code .env
```

Then copy the below into your new `.env` file

```env
MONGODB_URI=
PORT=5000
NODE_ENV=development
JWT_ACCESS_SECRET=
```
Then run the backend
```bash
cd server
npm i 
npm start
```
